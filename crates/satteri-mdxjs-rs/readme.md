# satteri-mdxjs

MDX-to-JavaScript compiler for Sätteri, a high-performance Markdown and MDX processor. A fork of [mdxjs-rs](https://github.com/wooorm/mdxjs-rs) by Titus Wormer, adapted to use oxc and pulldown-cmark.

> [!NOTE]
> **Work in progress** — this crate is under active development.
> See the [Contributing Guide](../../CONTRIBUTING.md) if you want to get involved!
